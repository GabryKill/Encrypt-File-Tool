<h1 align="center">Encryption File Tool</h1>

This is a tool to encrypt files with the default key editable in the vb.net project by rebuilding the little program with [Visual Studio editor for vb.net](https://visualstudio.microsoft.com/it/downloads/)

## Usage
- Navigate to ```\EncryptFile\EncryptFile\bin\Debug\net6.0-windows```
- Click to ```EncryptFile.exe```

## Importants
You can customize the program your way
